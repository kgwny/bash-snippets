# メタ文字
# * : 0文字以上の任意文字列にマッチ
# ?	: 1文字の任意文字にマッチ
# ~ : ホームディレクトリ
# #	: コメント
# \ : メタ文字を無効化する
# $ : 変数展開($var)
# " : 文字列 ""内に変数$varを記述した場合、変数展開が行われる
# ' : 文字列 ''内に変数$varを記述した場合、変数展開は行われない
# ` : コマンド実行結果参照 `command`
# ! : ヒストリ参照 !number
# ; : コマンド区切り文字 command1 ; command2
# | : コマンド実行結果を次のコマンドに引き渡す command1 | command2
# < : リダイレクト受信 command < file
# > : リダイレクト送信 command > file
# & : コマンドをバックグランド実行 command &
# (): コマンドをグループ化((command1; command2))
# []: テストコマンド
# {}: 変数展開 (${var})
