# 制御文字
# ^  : キャレット 文の先頭
# $  : ダラー 文の末尾
# \  : バックスラッシュ エスケープ文字

# [] : 大括弧 大括弧内のいずれかの文字
# |  : パイプ パイプで区切られた、いずれかの文字列

# .  : ピリオド 任意の1文字
# *  : アスタリスク 直前の文字を0回以上連続で繰り返す
# +  : プラス 直前の文字を1回以上連続で繰り返す
# ?  : クエスチョンマーク 直前の文字を0回もしくは1回だけ繰り返す

# 中括弧
# {n}   : 指定回数の繰り返し
# {n,}  : 指定回数以上の繰り返し
# {n,m} : 指定回数範囲内の繰り返し

# *?  :	0回以上の繰り返し
# +?  :	1回以上の繰り返し
# ??  :	0回か1回の繰り返し
# {}? :	指定範囲内での最短一致

# ()   : 小括弧 複数の文字で構成される文字列
# ()+  : 括弧内の文字列が連続で繰返り返される文字列
# (.*) : 置換文字列
